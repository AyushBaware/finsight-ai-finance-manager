# 🚀 FinSight – AI-Powered Personal Finance Manager

*AI-driven web application for expense tracking, financial analytics, and smart investment insights*  
*Built using React + Firebase + PWA architecture*

---

## 🔗 Links
- **Live Demo (Vercel):** Coming Soon  
- **GitHub Repo:** https://github.com/AyushBaware/finsight-ai-finance-manager  

---

## 🧠 Overview
- Track daily expenses efficiently  
- Analyze spending behavior with real-time data  
- Make data-driven financial decisions  
- Get AI-powered financial insights and recommendations  

---

## ✨ Key Features

### 💰 Expense Management
- Quick expense logging *(manual + voice input)*  
- Automatic expense categorization  
- Search and filter expenses  
- Real-time sync using Firestore  

### 📊 Analytics & Insights
- Dashboard with income, expenses, and savings rate  
- Category-wise expense breakdown  
- Weekly spending analysis  
- AI-based expense pattern insights  

### 🤖 AI Financial Assistant
- Personalized financial recommendations  
- Risk-based investment suggestions  
- Wealth projection and investment simulation  
- Financial goal planning and tracking  

### 📱 PWA Experience *(Key Highlight)*
- Installable web app *(works like mobile app)*  
- Offline support using IndexedDB  
- Fully responsive *(mobile + desktop)*  

---

## 🛠 Tech Stack
- **Frontend:** React 19, Vite  
- **Backend:** Firebase *(Authentication + Firestore)*  
- **Styling:** Tailwind CSS  
- **Charts:** Recharts  
- **Other:** Axios, React Router, Vite PWA Plugin  

---

## ⚙️ Getting Started

```bash
git clone https://github.com/AyushBaware/finsight-ai-finance-manager.git
cd finsight-ai-finance-manager
npm install
npm run dev
```
App runs at: http://localhost:5173

## 🔐 Authentication & Data

- 🔑 **Google Sign-In:** Firebase Authentication  
- ☁️ **Database:** Secure real-time database *(Firestore)*  
- 📦 **Offline Support:** Persistence with automatic sync
